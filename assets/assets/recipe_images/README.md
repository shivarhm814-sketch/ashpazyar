# راهنمای بارگذاری عکس دستور پخت‌ها

برای اینکه عکس واقعی هر غذا به‌جای ایموجی نمایش داده بشه:

1. یک عکس با کیفیت مناسب (ترجیحاً افقی، حداقل ۸۰۰ در ۴۵۰ پیکسل) برای هر غذا پیدا کن.
2. عکس رو دقیقاً با همون اسمی که جلوی هر غذا توی جدول زیر نوشته شده ذخیره کن (پسوند jpg یا png).
3. فایل رو توی همین پوشه (assets/recipe_images) کپی کن — کنار همین README، نه توی زیرپوشه.
4. به Claude بگو دوباره build و deploy کنه.

اگه برای غذایی عکس نذاری، همون ایموجی فعلی نمایش داده می‌شه — هیچ‌چیزی خراب نمی‌شه.
